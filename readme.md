# This is a voice chat agent
```
Requirements:
For main-chat.py:
https://docs.langchain.com/oss/python/langchain/quickstart
1. To install the LangChain package:
    command: pip install -qU langchain "langchain[anthropic]" 
2. Set up a Claude (Anthropic) account and get an API key
3. Set the ANTHROPIC_API_KEY environment variable in your terminal

For main-ollama-chat.py:
https://docs.langchain.com/oss/python/integrations/chat/ollama
1. The LangChain Ollama integration lives in the langchain-ollama package:
    command: pip install -qU langchain-ollama
```
